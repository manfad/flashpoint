# Flashpoint for VS Code

Agent Session sidebar for [flashpoint](https://github.com/manfad/flashpoint):
anchors sealed around every AI-agent turn, native diffs per anchor, and
one-click **timetravel** back to any safe point. Nothing forks until you
change something after a timetravel — and no timeline is ever lost.

Requires the `fp` binary (`cargo install --path .` from the flashpoint repo,
or set `flashpoint.binary`).
